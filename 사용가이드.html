1. 시작하기 — 접속 & 로그인

접속 방법

  1. Chrome 또는 Edge 브라우저에서 https://netformrnd.github.io/sq-biz/ 접속
  2. 즐겨찾기(★) 등록 권장
  3. 모바일에서도 접속 가능하나, 일부 기능(엑셀 붙여넣기 등)은 PC 권장

로그인

  1. 관리자로부터 받은 아이디 / 비밀번호 입력
  2. 로그인 성공 시 대시보드 화면으로 이동
  3. 비밀번호 변경: 좌측 하단 [내 계정] → 비밀번호 변경
보안 안내
• 비밀번호 5회 연속 실패 시 10분간 계정 잠김
• 30분간 활동이 없으면 자동 로그아웃
• 초기 비밀번호는 반드시 변경하여 사용

3. 세금계산서 발행 요청 직원

절차

  1. 좌측 메뉴 [세금계산서 → 발행 요청] 클릭
  2. 거래처 사업자등록증 화면캡쳐 준비
    • Windows: ⊞ Win + Shift + S 로 영역 캡쳐
    • Mac: ⌘ + Control + Shift + 4
  3. 캡쳐 후 발행 요청 페이지에서 Ctrl + V 로 붙여넣기
    • OCR 자동 인식 → 상호·사업자번호·대표자·주소 자동 입력
  4. OCR 오인식 항목 직접 수정 (거래처명/번호 꼭 확인)
  5. 공급가액 입력 → 세액(10%)·합계 자동 계산
  6. 발행 사유, 프로젝트명, 이메일 입력
  7. 필요 시 계약서 파일 첨부 (드래그 또는 클릭)
  8. [발행 요청] 버튼 클릭 → 관리자에게 알림
📋 나의 요청현황에서 확인 가능한 상태
  • 요청 — 접수됨, 관리자 검토 대기
  • 검토중 — 관리자가 검토 시작
  • 발행완료 ✅ — 세금계산서 실제 발행됨
  • 반려 ❌ — 반려사유 확인 후 재요청
주의사항
  • 사업자등록증 파일 클릭 업로드는 지원 중단 — Ctrl+V 화면캡쳐만 사용
  • OCR 인식 오류가 잦으면 수동으로 각 항목 입력 후 요청
  • 발행 요청 후 수정이 필요하면 관리자에게 반려 요청

반려된 요청 재처리

  1. [나의 요청현황] → 반려 건 확인 → 반려사유 확인
  2. 새로 [발행 요청]에서 수정된 내용으로 재요청

4. 세금계산서 발행 관리 관리자

요청 검토 & 발행 처리

  1. [세금계산서 발행] 메뉴 → 상태 탭 [요청] 선택
  2. 발행 대기 건의 [검토하기] 버튼 클릭 (상태 자동 "검토중"으로 변경)
  3. 상세 팝업에서 거래처 정보·금액·첨부 확인
  4. 위하고 등 실제 세금계산서 발행 완료 후 버튼 클릭:
    • [발행완료 처리] — 발행일 기록, 요청자에게 알림
    • [반려] — 사유 입력 → 요청자에게 알림

발행내역 조회 (직원 포함)

직원도 조회 모드로 접근 가능. 다음 정보를 확인할 수 있습니다:

  • 요청번호 · 발행일 · 입금일 · 입금처
  • 거래처명 · 발행사유 · 합계금액 · 상태
  • 상세보기 👁️ 버튼으로 전체 내용 확인

필터 & 검색

기능설명
🗓️ 당일 탭오늘 요청/발행된 건만 표시
요청 / 검토중 / 발행완료 / 반려상태별 필터
✅ 완료 숨기기매칭 완료된 발행건 숨김 → 미입금/미처리 집중
🔍 검색요청번호/거래처/사업자번호/발행사유/입금처/금액 통합 검색
📋 붙여넣기홈택스/위하고 매출 목록 일괄 등록 (자동 헤더 인식)
+ 개별 등록수동으로 세금계산서 1건 추가

매칭 상태 표시

  • 🔵 파란 배경: 단건 매칭 (입금 1건)
  • 🟢 초록 배경 · ✅ 완료: 다건 매칭 + 합계 일치
  • 🟠 주황 배경 · (부분 XX%): 부분입금 — 추가 입금 필요
  • ⚫ 미입금: 아직 입금 없음

5. 입금내역 & 매칭 전체

5-1. 통장내역 업로드 (관리자)

  1. [입금내역] 메뉴 → 상단 [📊 은행/위하고 붙여넣기] 클릭
  2. 업로드 방법 (택1):
    • 파일 드래그: .xlsx / .xls 파일을 드롭존에 드래그
    • 파일 선택: 드롭존 클릭 후 파일 선택
    • 복사·붙여넣기: 엑셀에서 복사 후 Ctrl+V
  3. 지원 포맷:
    • 기업은행 입출식예금 (헤더 없음 자동 인식)
    • 위하고 (월/일 | 적요 | 입금액 | 출금액 | 잔액)
    • 은행 일반 (거래일시 | 출금 | 입금 | 잔액 | 거래처명)
  4. 자동 분석 후 입금 / 송금 분리 미리보기 확인
  5. 필요 시 [⚠️ 기존 입금/송금 모두 삭제 후 업로드] 체크
    이 옵션은 되돌릴 수 없습니다. 테스트 데이터 정리 시에만 사용.
  6. [선택 항목 등록] 클릭

5-2. 미매칭 입금 확인 → 매칭하기

  1. 상단 [미매칭] 필터로 매칭 필요한 건 확인
  2. 해당 입금건의 👁️ 상세/매칭 버튼 클릭
  3. 모달에서 🏢 거래처 추천 자동 제공
    • ✓ 선택 시 거래처 지정 + 사업자번호 자동 기입
    • ✕ 클릭 시 해당 추천 영구 숨김
    • 검색 박스에서 직접 입력 가능
  4. 🔗 선발행 건 매칭 섹션에서 추천 중 [매칭] 클릭 또는 드롭다운에서 직접 선택
  5. 확인 팝업에서 완전/부분/초과 여부 안내 확인 → 확인

5-3. 다건 입금 → 1건 세금계산서 (부분입금) 매칭

사례: 공급가액과 부가세가 별도로 2번 입금된 경우

  1. 첫 번째 입금 매칭 → 확인창에 "부분 매칭 (남은금액 ...)" 표시
  2. 두 번째 입금의 상세 모달 진입
  3. 추천 목록에 해당 세금계산서가 🟠 "부분입금 1건"으로 노출됨
    • 남은 금액과 일치하면 유사도 최고점(60) + ✅ 완전 매칭 예고
  4. [추가매칭] 버튼 클릭 → 완료

세금계산서 발행 페이지에서 "외 1건 ✅ 완료"로 표시됩니다.

5-4. 매칭 해제 / 전체 초기화

  • 개별 해제: 상세 모달 → 매칭된 항목의 [🔗 매칭 해제] 클릭 (해당 건만)
  • 전체 초기화: 페이지 상단 [🗑️ 전체 초기화] — 입금 + 송금 전체 삭제
    • 이중 확인: 네이티브 confirm + "삭제" 단어 입력 필수

6. 연차 신청 전체

6-1. 연차 신청하기

  1. 좌측 메뉴 [연차 신청] 클릭
  2. 달력에서 신청할 날짜 클릭 → 팝업 표시
  3. 연차 종류 선택:
    • 연차 — 1일 차감 (종일)
    • 반차 — 0.5일 차감 (4시간)
    • 반반차 — 0.25일 차감 (2시간)
  4. 반차/반반차 선택 시 시간 프리셋 또는 커스텀 시간 지정
    근무시간 8:30 ~ 17:30 기준 / 점심시간 12:00 ~ 13:00 자동 제외
    반차는 4시간, 반반차는 2시간으로 고정 차감
  5. 사유 입력 (선택) → [신청] 클릭
  6. 관리자 승인 대기 (우측 사이드에 대기 건수 표시)

6-2. 취소 요청

이미 승인된 연차를 취소하려면:

  1. 사이드 [📋 내 신청 내역]에서 해당 건 우측 [취소 요청] 클릭
  2. 취소 사유 입력 → 관리자 승인 대기
  3. 신청 직후(대기 중)라면 [신청 취소] 바로 가능

6-3. 관리자 기능

  • ⏳ 승인대기 — 신규 신청 / 취소 요청 일괄 승인·반려
  • 👥 팀원 연차 관리
    • 기본 연차 개수 편집 (15일 기본)
    • 포상 연차 추가 (사유 포함)
    • 무제한 연차 설정 가능
    • 팀원별 "연차 사용 대상 추가/제외" 토글
  • 📊 리포트 — 엑셀(.xlsx) 내보내기 (연차현황 + 사용일자상세 2시트)
  • 📥 데이터 이관 — 기존 연차 시스템에서 이름 기반 자동 매칭 이관

6-4. 달력 색상

  • 👤 김영성 — 파란색
  • 👤 신수진 — 핑크색
  • 대기중(점선 테두리) / 승인완료(실선) 자동 구분
  • 공휴일 빨강색 자동 표시 (클릭 불가)

7. 문서보관 전체

자동 저장 (세금계산서 요청 시)

발행 요청 시 첨부한 사업자등록증·계약서가 자동으로 문서보관으로 이관되어 거래처명별로 분류됩니다.

수동 업로드 (관리자)

  1. [문서보관] 메뉴 → 상단 [+ 문서 업로드] 클릭
  2. 사업자명 입력 (공통)
  3. 카테고리별 파일 선택 (동시 선택 가능):
    • 📄 사업자등록증 (여러 파일)
    • 📑 계약서 (여러 파일)
    • 📎 기타 문서
  4. 업로드 → 카테고리별 건수 자동 분류 표시

주요 기능

  • 🔍 사업자명/사업자번호/파일명 검색
  • 📅 등록일 기준 날짜 필터
  • ✏️ 파일명 변경 / 👁️ 미리보기 / 💾 다운로드 / 🗑️ 삭제(관리자)
  • 설정 메뉴에서 전체 문서 ZIP 일괄 다운로드 지원

8. 업무 체크리스트 관리자

주기별 체크리스트

  • 일일 — 매일 반복 업무 (ex: 통장 확인)
  • 월간 — 월별 업무 (ex: 급여, 4대보험)
  • 반기 / 연간 — 종합소득세, 결산 등

30개 이상 기본 항목이 내장되어 있습니다. 체크 시 담당자/시간이 자동 기록됩니다.

효율 팁

  • 매칭 시 금액 완전일치되면 유사도 90점 이상 → 최상단에 추천됨
  • 부분입금 건은 남은 금액만 추가 매칭하면 됨
  • 엑셀 붙여넣기 시 반드시 헤더 행 포함해서 복사 (컬럼 자동 인식)
  • 상태·월·완료숨기기 필터 조합 사용하면 목록이 간결해짐
  • 세금계산서 발행 테이블 내용이 길 때는 마우스 호버로 전체 내용 확인
  • 당일 탭 + 완료숨기기 체크 → 오늘 처리해야 할 건만 집중 조회
