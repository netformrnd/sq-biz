📢 업무관리 시스템이 도입되었습니다

기존 엑셀·종이로 관리하던 세금계산서 발행 요청 · 입금내역 매칭 · 연차 · 문서보관 · 업무 체크리스트를 하나의 웹 시스템으로 통합하였습니다. 오늘부터 모든 업무는 이 시스템을 통해 진행해 주시기 바랍니다.

🔗 접속: https://netformrnd.github.io/sq-biz/ · 즐겨찾기 등록 권장 · 권장 브라우저: Chrome / Edge

⭐ 꼭 알아야 할 3가지

1️⃣ 세금계산서 발행 요청

  • 사업자등록증 화면캡쳐 (⊞Win+Shift+S)
  • 요청 페이지에서 Ctrl+V → OCR 자동 인식
  • 공급가액만 입력 → 세액·합계 자동계산
  • 관리자 승인 후 발행 진행

2️⃣ 입금내역 업로드 & 매칭

  • 통장 엑셀 파일 드래그 업로드 (기업은행/위하고/일반은행)
  • 👁️ 상세 버튼 → 거래처 자동 추천 ✓선택
  • 🔗 선발행 건 매칭 → 세금계산서와 연결
  • 부분입금(공급가액+부가세 분리)도 지원

3️⃣ 연차 신청

  • 달력에서 날짜 클릭 → 연차/반차/반반차 선택
  • 근무시간 08:30~17:30, 점심 12:00~13:00 자동 반영
  • 반차 = 4시간 / 반반차 = 2시간 차감
  • 관리자 승인 후 확정 (사이드에서 내 잔여 확인)
⚠️ 꼭 지켜주세요  ·  본인 계정으로만 로그인 (타인 계정 공유 금지, 모든 작업은 이력 기록)  ·  사업자등록증 첨부는 화면캡쳐 + Ctrl+V로만 진행 (파일 업로드 미지원)  ·  초기 비밀번호는 첫 접속 시 즉시 변경 (내 계정 메뉴)  ·  30분 미활동 시 자동 로그아웃

내부 업무용

📝 세금계산서 발행 요청 (직원 → 관리자) 직원

👤 직원 — 요청 올리기

  1. [세금계산서 → 발행 요청] 메뉴 접속
  2. 거래처 사업자등록증 화면캡쳐
  3. 페이지에서 Ctrl+V 붙여넣기 → OCR 자동 입력
  4. 오인식 항목 있으면 직접 수정 (상호/사업자번호 필수 확인)
  5. 공급가액 입력 (세액/합계 자동)
  6. 발행 사유·프로젝트명·이메일 입력
  7. 계약서가 있으면 드래그로 첨부 (선택)
  8. [발행 요청] 클릭 → 완료
  9. [나의 요청현황]에서 상태 추적 가능

👔 관리자 — 검토·발행

  1. [세금계산서 발행] → 🗓️ 당일 탭에서 새 요청 확인
  2. [검토하기] 클릭 → 상세 팝업 열림
  3. 거래처·금액·첨부 확인
  4. 위하고/홈택스에서 실제 세금계산서 발행
  5. 시스템에서 [발행완료 처리] 클릭
  6. 잘못된 요청은 사유 입력 후 [반려]
  7. 필요 시 거래처명·이메일 수정 (이름 클릭)

💰 입금내역 업로드 & 매칭 관리자 조회

📥 통장내역 일괄 업로드

  1. [입금내역] → [📊 은행/위하고 붙여넣기] 클릭
  2. 엑셀 파일 드래그&드롭 (또는 복사붙여넣기)
  3. 자동 파싱 → 입금/송금 분리 미리보기
  4. 체크박스로 필요 항목만 선택
  5. 기존 데이터 지울 경우 초기화 체크
  6. [선택 항목 등록] 클릭

🔗 세금계산서와 매칭하기

  1. 미매칭 입금건의 👁️ 상세 버튼 클릭
  2. 🏢 거래처 추천 → ✓ 선택 (사업자번호 자동 채움)
  3. 🔗 선발행 매칭 → 추천 목록에서 [매칭] 클릭
  4. 완전매칭/부분매칭 안내 확인
  5. 공급가액/부가세 분리 입금도 추가매칭으로 합산

🗓️ 연차 신청 방법 전체

  1. [연차 신청] 메뉴 접속
  2. 달력에서 원하는 날짜 클릭
  3. 연차(1일) / 반차(0.5일) / 반반차(0.25일) 선택
  4. 반차/반반차는 시간 프리셋 선택 가능
  5. 사유 입력(선택) → [신청]
  6. 관리자 승인 후 확정 · 잔여 연차는 사이드에서 확인
  7. 승인 후 취소하려면 내 신청 내역에서 [취소 요청]

📁 문서보관 전체

  • 발행 요청 시 첨부한 사업자등록증·계약서 자동 저장
  • 거래처명별 가나다순 자동 정렬
  • 별도 업로드: 📄 사업자등록증 / 📑 계약서 / 📎 기타
  • 🔍 사업자명/사업자번호/파일명 검색 가능
  • 👁️ 미리보기, 💾 다운로드, ✏️ 이름변경 지원

⌨️ 자주 쓰는 단축키 & 팁

⌨️ 단축키

  • Ctrl+K 전역 검색
  • Ctrl+V 사업자등록증/엑셀 붙여넣기
  • Esc 팝업 닫기
  • 우클릭 → 빠른 메뉴

💡 효율 팁

  • 엑셀 복사 시 헤더 행 포함
  • 🗓️ 당일 + ✅ 완료숨기기 조합 → 처리할 건만 집중
  • 부분입금은 남은금액만 추가매칭
  • 잘린 내용은 마우스 호버로 전체 보기

🛠️ 문제 해결

  • 변경 반영 안될 때: Ctrl+Shift+R
  • 비번 분실: 관리자에게 초기화 요청
  • OCR 오류: 수동 입력
  • 매칭 취소: [🔗 매칭 해제] 후 재매칭
