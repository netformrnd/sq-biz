netformrnd.github.io/sq-biz

📌 핵심 업무 흐름 — 세금계산서 발행부터 입금 확인까지

1
발행 요청👤 요청자(직원)
📝 세금계산서 → 발행 요청
  • 거래처 사업자등록증 화면캡쳐 (⊞Win+Shift+S) → 페이지에 Ctrl+V
  • OCR로 상호·사업자번호·대표자 자동 입력 → 공급가액만 넣으면 세액·합계 자동계산
  • 사유 입력 후 [발행 요청] 클릭 → 관리자에게 알림
2
검토 & 발행👔 경리 관리자
🧾 세금계산서 발행
  • 🗓️ 당일 탭에서 신규 요청 확인 → [검토하기]
  • 위하고/홈택스에서 실제 발행 후 시스템에서 [발행완료 처리] (또는 사유 입력하여 반려)
  • 발행된 내역은 직원도 조회 모드로 확인 가능
3
입금내역 업로드👔 경리 관리자
💰 입금내역
  • [📊 은행/위하고 붙여넣기] → 통장 엑셀 파일 드래그 & 드롭 (또는 복사·붙여넣기)
  • 기업은행/위하고 형식 자동 인식 · 입금/송금 자동 분리
  • 미리보기 확인 → [선택 항목 등록]
4
거래처 지정 & 매칭👔 경리 관리자
💰 입금내역 → 👁️ 상세
  • 🏢 거래처 추천 목록에서 ✓ 선택 (사업자번호 자동 입력)
  • 🔗 선발행 매칭 → 세금계산서와 연결 · 유사도 높은 순 자동 추천
  • 공급가액·부가세 분리 입금 건은 추가매칭으로 합산 처리
5
문서 자동 보관🤖 시스템 자동
📁 문서보관
  • 발행 요청 시 첨부한 사업자등록증·계약서가 거래처명별로 자동 저장
  • 가나다순 자동 정렬 · 검색 · 미리보기 · 다운로드 지원
  • 필요 시 [+ 문서 업로드]로 사업자등록증/계약서/기타 직접 등록

🔧 그 외 기능

🗓️ 연차 신청

연차 신청 달력에서 날짜 클릭 → 연차/반차/반반차 선택 → 관리자 승인. 잔여 연차는 사이드에서 실시간 확인.

🔍 전역 검색

Ctrl+K로 요청번호·거래처·금액 통합 검색. 각 메뉴의 당일 탭 + 완료 숨기기 조합으로 처리할 건만 집중.

⚠️ 주의사항

본인 계정으로만 로그인 (공유 금지). 사업자등록증은 화면캡쳐 + Ctrl+V만 사용. 초기 비밀번호는 즉시 변경.
