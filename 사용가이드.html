📌 업무 흐름

1
발행 요청 👤 직원 📝 세금계산서 → 발행 요청
사업자등록증 캡쳐 → Ctrl+V OCR 자동입력 → 공급가액 입력 → 발행 요청
2
검토 & 발행 👔 관리자 🧾 세금계산서 발행
🗓️ 당일 탭 확인 → 검토하기 → 위하고/홈택스 발행 → 발행완료 처리
3
입금내역 업로드 👔 관리자 💰 입금내역
통장 엑셀 파일 → 📊 드래그 업로드 → 자동 분석 → 선택 항목 등록
4
거래처 지정 & 매칭 👔 관리자 💰 입금내역 → 👁️ 상세
🏢 거래처 추천 ✓ → 🔗 선발행 매칭 → 완료 (부분입금도 지원)
5
문서 자동 보관 🤖 시스템 📁 문서보관
첨부파일 → 거래처명별 자동 분류 → 언제든 검색·다운로드
